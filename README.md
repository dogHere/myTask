# 问题总结

## 未按照文档给定要求实现

略

## git问题

1. `git commit`内容太简略，要写明白git的操作是啥。
2. `git` 不要把`.mvn`类似这样的东西添加进来。
3. 一定记得写一个readme文件，注释好你写的内容是啥，应该怎么使用。

## 编码习惯问题

1. 类命名不要随便写，比如`Task1`这样的名字，尽量写和业务相关的名字。
2. 注意方法私有化，某些地方应该加上`private`。
3. 其他见代码内部`todo`注释

## 异常处理问题

建议了解下java异常处理的`基本原则`。不要试图全局处理异常，全局处理等价于不处理异常。
